# Lottery 규칙

1. +3번째 블록해쉬의 첫 두글자 맞추기 "0xab..."
    - 유저가 던진 트랜잭션이 들어가는 블록 + 3의 블록해쉬와 값을 비교
2. 팟 머니
    - 결과가 나왔을 때만 유저가 보낸 돈을 팟 머니에 쌓기
    - 여러명이 맞추었을 때는 가장 먼저 맞춘 사람이 팟머니를 가져간다.
    - 두 글자 중하나만 맞추었을 때는 보낸 돈을 돌려준다.
    - 결과 값을 검증할 수 없을 때는 보낸 돈을 돌려준다.
